# CupomMaster - Frontend

Sistema completo de gestão de cupons de desconto desenvolvido com Angular 19 e Tailwind CSS 3.

##  Tecnologias

- Angular 19 - Framework web
- Tailwind CSS 3 - Framework de estilos
- TypeScript - Linguagem de programação
- RxJS - Programação reativa

```
npm install
npm start
```

Acesse: http://localhost:4200
